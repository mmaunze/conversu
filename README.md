# Website Conversu - Construções Verdes e Sustentáveis

Bem-vindo ao repositório do website da empresa Conversu - Construções Verdes e Sustentáveis. Este projeto foi desenvolvido do zero para oferecer uma plataforma informativa e administrativa, abrangendo desde informações sobre a empresa até um sistema completo de administração de conteúdo.

## Funcionalidades

### Parte do Conteúdo

- **Página Inicial**: Apresenta uma visão geral da empresa, incluindo sua missão, visão e valores.
- **Portfólio**: Exibe projetos concluídos pela empresa, destacando suas especializações em construções verdes e sustentáveis.
- **Blog**: Contém artigos educativos e informativos sobre construção verde, sustentabilidade e inovações no setor.

### Parte de Administração de Conteúdo

- **Gestão de Utilizadores**: Cadastro e gerenciamento de usuários com diferentes níveis de permissão.
- **Categorias de Conteúdo**: Criação, edição e exclusão de categorias para organizar os artigos do blog.
- **Atualização do Portfólio**: Adição, edição e remoção de projetos no portfólio da empresa.
- **Gestão de Artigos**: Criação, edição e publicação de novos artigos no blog utilizando um editor WYSIWYG personalizado.
- **Publicação de Conteúdo**: Interface intuitiva para gerenciar e publicar conteúdo sem a necessidade de habilidades técnicas avançadas.

## Tecnologias Utilizadas

### Frontend

- **HTML, CSS**: Estruturação e estilização do conteúdo.
- **JavaScript (com jQuery e Ajax)**: Implementação de interações dinâmicas e requisições assíncronas.
- **Bootstrap**: Framework para criação de interfaces responsivas e componentes de interface.

### Backend

- **PHP**: Linguagem de programação para lógica de negócios e interação com o banco de dados.
- **MySQL**: Sistema de gerenciamento de banco de dados relacional para armazenamento seguro de dados.

### Editor de Texto

- **WYSIWYG Personalizado**: Ferramenta para criação e edição de conteúdo diretamente no navegador, simplificando o processo de publicação de artigos.

## Instalação e Configuração

1. **Clone o Repositório:**
   ```bash
   git clone https://github.com/mmaunze/conversu.git
   cd conversu
   ```

2. **Configuração do Banco de Dados:**
   - Importe o arquivo SQL fornecido para criar a estrutura inicial do banco de dados MySQL.
   - Configure as credenciais de acesso ao banco de dados no arquivo de configuração (`config.php`).

3. **Configuração do Ambiente de Desenvolvimento:**
   - Certifique-se de ter um servidor web (como Apache) e PHP instalados no seu ambiente de desenvolvimento.
   - Configure o servidor web para apontar para o diretório raiz do projeto.

4. **Execução do Projeto:**
   - Abra o navegador e acesse o URL do seu servidor local para iniciar o website.
